## Package Overview

The CommandOSS TypeScript SDKs consist of three main packages:

| Package | Purpose | Required |
|---------|---------|----------|
| `@cmdoss/walrus-site-builder` | Core SDK for Walrus + Sui integration | ‚úÖ Yes |
| `@cmdoss/walrus-site-builder-react` | React hooks and components | üî∂ For React apps |
| `@cmdoss/file-manager` | Browser file system with ZenFS | üî∂ For React apps |

## Prerequisites

Before installing the SDKs, ensure you have:

- **Node.js** 18.0 or higher
- **npm** 9.0+ / **pnpm** 8.0+ / **yarn** 1.22+
- **TypeScript** 5.0 or higher (recommended)
- A Sui wallet extension (for blockchain interactions)

## Environment Setup

### TypeScript Configuration

Ensure your `tsconfig.json` includes these settings for optimal compatibility:

```json
{
  "compilerOptions": {
    "lib": ["ES2021", "DOM"],
    "module": "esnext",
    "moduleResolution": "bundler",
    "target": "ES2022",
    "strict": true,
    "skipLibCheck": true,
    "resolveJsonModule": true,
    "esModuleInterop": true
  }
}
```

### Wallet Integration

You'll need to set up wallet integration for Sui blockchain transactions:

```typescript
import { createNetworkConfig, SuiClientProvider, WalletProvider } from '@mysten/dapp-kit';
import { getFullnodeUrl } from '@mysten/sui/client';
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';

const { networkConfig } = createNetworkConfig({
  localnet: { url: getFullnodeUrl('localnet') },
  testnet: { url: getFullnodeUrl('testnet') },
  mainnet: { url: getFullnodeUrl('mainnet') },
});

const queryClient = new QueryClient();

function App() {
  return (
    <QueryClientProvider client={queryClient}>
      <SuiClientProvider networks={networkConfig} defaultNetwork="testnet">
        <WalletProvider autoConnect>
          {/* Your app components */}
        </WalletProvider>
      </SuiClientProvider>
    </QueryClientProvider>
  );
}
```

### Walrus Client Setup

Configure the Walrus client for decentralized storage:

```typescript
import { WalrusClient } from '@mysten/walrus';

const walrusClient = new WalrusClient({
  aggregatorUrl: 'https://aggregator.walrus-testnet.walrus.space',
  publisherUrl: 'https://publisher.walrus-testnet.walrus.space'
});
```

## Verification

Test your installation with this simple verification script:

```typescript
import { WalrusSiteBuilderSdk, getSHA256Hash, sha256ToU256 } from '@cmdoss/walrus-site-builder';

// Verify the SDK can be imported and basic functions work
async function verifyInstallation() {
  try {
    const testContent = new TextEncoder().encode('Hello World');
    const hash = await getSHA256Hash(testContent);
    const hashU256 = sha256ToU256(hash);
    
    console.log('‚úÖ Installation verified!');
    console.log(`Hash: ${Array.from(hash).map(b => b.toString(16).padStart(2, '0')).join('')}`);
    console.log(`Hash U256: ${hashU256.toString()}`);
  } catch (error) {
    console.error('‚ùå Installation verification failed:', error);
  }
}

verifyInstallation();
```

### Version Conflicts

If you encounter version conflicts:

```bash
# Check installed versions
npm ls @cmdoss/walrus-site-builder

# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install
```
