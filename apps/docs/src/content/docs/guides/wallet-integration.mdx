The Site Builder SDK integrates seamlessly with Sui wallets through the Sui dApp Kit, providing a smooth user experience for connecting wallets, managing accounts, and signing transactions. This guide covers everything you need to know about wallet integration.

## Overview

Wallet integration provides:
- **Multi-wallet support** - Connect to various Sui wallets
- **Account management** - Handle multiple accounts and addresses
- **Transaction signing** - Sign and execute blockchain transactions
- **Balance checking** - Monitor Sui and WAL token balances
- **Network switching** - Support for testnet and mainnet

## Prerequisites

Before integrating wallets, ensure you have:

- **Sui dApp Kit** installed and configured
- **Wallet adapters** for your target wallets
- **Network configuration** for Sui and Walrus
- **Proper error handling** for wallet operations

## Basic Setup

### 1. Install Dependencies

```bash
npm install @mysten/dapp-kit @mysten/sui @mysten/walrus
```

### 2. Configure dApp Kit

```typescript
import { SuiClientProvider, WalletProvider } from '@mysten/dapp-kit';
import { getFullnodeUrl } from '@mysten/sui/client';
import { createNetworkConfig } from '@mysten/dapp-kit';

// Network configuration
const networkConfig = createNetworkConfig({
  devnet: { url: getFullnodeUrl('devnet') },
  testnet: { url: getFullnodeUrl('testnet') },
  mainnet: { url: getFullnodeUrl('mainnet') },
});

// Wallet configuration
const walletConfig = {
  wallets: [
    // Add your preferred wallet adapters
  ],
};
```

### 3. Provider Setup

```typescript
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';

const queryClient = new QueryClient();

function App() {
  return (
    <QueryClientProvider client={queryClient}>
      <SuiClientProvider networks={networkConfig} defaultNetwork="testnet">
        <WalletProvider
          wallets={walletConfig.wallets}
          autoConnect={true}
        >
          <YourApp />
        </WalletProvider>
      </SuiClientProvider>
    </QueryClientProvider>
  );
}
```

## Balance Management

### Sui Balance

```typescript
import { useSuiBalance } from '@/hooks/useSuiBalance';

function SuiBalance() {
  const { balance, isLoading, error } = useSuiBalance();

  if (isLoading) return <p>Loading balance...</p>;
  if (error) return <p>Error loading balance: {error.message}</p>;

  return (
    <div className="balance">
      <h3>Sui Balance</h3>
      <p>{balance.totalBalance} SUI</p>
      <p>Available: {balance.availableBalance} SUI</p>
    </div>
  );
}
```

### WAL Token Balance

```typescript
import { useWalBalance } from '@/hooks/useWalBalance';

function WalBalance() {
  const { balance, isLoading, error } = useWalBalance();

  if (isLoading) return <p>Loading WAL balance...</p>;
  if (error) return <p>Error loading WAL balance: {error.message}</p>;

  return (
    <div className="wal-balance">
      <h3>WAL Balance</h3>
      <p>{balance} WAL</p>
    </div>
  );
}
```
