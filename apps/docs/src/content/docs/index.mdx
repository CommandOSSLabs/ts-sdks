## Basic Usage Example

Here's how to deploy your first decentralized website:

```typescript
import { WalrusSiteBuilderSdk, getSHA256Hash, sha256ToU256 } from '@cmdoss/site-builder'

// Initialize the SDK
const sdk = new WalrusSiteBuilderSdk(
  walrusClient,
  suiClient, 
  activeAccount,
  signAndExecuteTransaction
)

// Prepare your assets
const content = '<h1>Hello Decentralized Web!</h1>'
const contentBytes = new TextEncoder().encode(content)
const hash = await getSHA256Hash(contentBytes)

const assets = [{
  path: '/index.html',
  content: contentBytes,
  hash,
  hashU256: sha256ToU256(hash)
}]

// Deploy in 5 simple steps
const deployFlow = sdk.deployFlow(assets)
await deployFlow.prepareAssets()     // 1. Register blobs
await deployFlow.uploadAssets(5)     // 2. Upload to Walrus (5 epochs)
await deployFlow.certifyAssets()     // 3. Certify on Sui
const siteId = await deployFlow.updateSite()  // 4. Create/update site

console.log(`Site deployed! ID: ${siteId}`)
```

## Architecture Overview

Our SDKs are built with a modular architecture:

- **[@cmdoss/site-builder]** - Core SDK with Walrus + Sui integration
